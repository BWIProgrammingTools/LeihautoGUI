# LeihautoGUI

Für unser Schulprojekt haben wir zwei GUIs einer Leihautofirma erstellt.
Für die Benutzung müssen zuerst einige Voreinstellungen getroffen werden, welche im LeihautofirmaGuide.pdf (auch in diesem Projekt angehängt) genauer beschrieben sind.

Das GUI aus Kundensicht wird über die Main-Methode im MainKunde.java oder im entsprechenden Git Ordner über KundenPortal.exe gestartet.

Das GUI aus Adminsicht wird über die Main-Methode im MainAdmin.java oder im entsprechenden Git Ordner über AdminPortal.exe gestartet.
